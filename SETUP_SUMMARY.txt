🎯 الخطوات التالية (يدوية):

  1️⃣  إعداد قاعدة بيانات
      └─ موصى به: Neon (https://neon.tech) - PostgreSQL مجاني
      └─ احصل على DATABASE_URL
      └─ حدّث schema.prisma إلى PostgreSQL
  
  2️⃣  رفع المشروع إلى GitHub
      └─ git init
      └─ git add .
      └─ git commit -m "Ready for Netlify"
      └─ git push
  
  3️⃣  ربط مع Netlify
      └─ https://app.netlify.com
      └─ Import from GitHub
      └─ الإعدادات تلقائية من netlify.toml ✅
  
  4️⃣  إضافة متغيرات البيئة في Netlify
      └─ DATABASE_URL = <your-database-url>
  
  5️⃣  تشغيل Migration والنشر
      └─ npx prisma db push
      └─ Deploy على Netlify
